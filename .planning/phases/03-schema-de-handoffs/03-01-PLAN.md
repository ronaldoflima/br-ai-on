  <files>lib/handoff.sh</files>

  <action>
Add new function handoff_thread_history() to lib/handoff.sh BEFORE the command dispatch block (before the `command="${1:?..."` line).

The function:
- Takes 1 required arg: thread_id
- Initializes empty array: `local results=()`
- Loops through all `$AGENTS_DIR/*/handoffs/{inbox,in_progress,archive}/` dirs
- For each HO-*.md file, greps for `^thread_id: $thread_id`
- If matched, extracts from, to, status, created via grep+sed
- Appends to results array as `"$created|$from|$to|$status"`
- If results empty, prints "No handoffs found for thread $thread_id" and returns 0
- Otherwise, printf+sort+while-read to output: `$created  $from -> $to  [$status]`

Then add to the case dispatch block (before the `*` catch-all):
  `thread-history) handoff_thread_history "$@" ;;`

Update usage comment at top to include: `handoff.sh thread-history <thread_id>`
